# Better Chat

Legacy Fabric mod by [LlamaLad7](https://github.com/LlamaLad7) which lets you customize the chat.

Options:

* Animated Chat (Messages slide up when received)

* Clear Chat Background

* Move the chat to where you want it

* Change its Scale
